# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Use [GitHub Private Security Advisories](https://github.com/mudrii/hom/security/advisories/new) to report a vulnerability privately. You will receive a response within 7 days.

Include as much of the following as possible:

- Description of the vulnerability
- Steps to reproduce
- Affected version(s)
- Potential impact
- Suggested fix (if any)

We will coordinate disclosure timing with you before publishing a fix.
